# Data Source

This project uses the **Our World in Data CO2 and Greenhouse Gas Emissions** dataset.

- **Source:** [Our World in Data](https://ourworldindata.org/co2-and-greenhouse-gas-emissions)
- **Raw data:** [GitHub Repository](https://github.com/owid/co2-data)
- **Direct URL:** https://raw.githubusercontent.com/owid/co2-data/master/owid-co2-data.csv

## Note
The dataset is loaded directly in the notebook via URL and does not need to be downloaded manually.
Data covers 1750–2024 across 200+ countries/regions.
